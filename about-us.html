About Dogen

Dogen is a code generator designed to target domain models, inspired by the domain driven design methodology. Our main objective is to automate the modeling process as much as possible: the user creates domain models using a supported UML tool and respecting a set of predefined restrictions; Dogen then uses the toolâ€™s diagram files to generate the source code representation. The generated code contains most of the services required from a typical C++ domain object such as serialisation, hashing, streaming and so on.
